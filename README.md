# Landing Page NovaCorp

Cette maquette fournit une landing page minimaliste et premium pour une activité de conseil digital (fictive). Les textes et images sont fictifs et peuvent être remplacés.

## Structure
- `index.html` : structure HTML principale
- `styles.css` : styles responsives mobile-first
- `script.js` : ouverture du menu mobile et animations de fade
- `assets/` : images SVG libres (logos, illustrations, avatars)

## Édition rapide
1. **Nom & activité** : remplacez les mentions "NovaCorp" et adaptez le contenu dans `index.html`.
2. **Couleurs** : modifiez les variables CSS dans `styles.css` (`--primary`, `--gray`...).
3. **Sections** : dupliquez/supprimez les blocs selon vos besoins (bénéfices, tarifs, FAQ...).
4. **Images** : remplacez les SVG du dossier `assets` par vos propres visuels optimisés.
5. **SEO** : mettez à jour `<title>`, `meta description` et le script JSON-LD dans `index.html`.
